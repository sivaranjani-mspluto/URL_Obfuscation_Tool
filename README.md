A simple Python-based tool to encode and decode URLs using **URL encoding** and **Base64**. Useful for cybersecurity learners and analysts to understand how attackers hide malicious links ‚Äî and how to reveal them.

---

## Features

- Obfuscate any URL using:
  - URL Encoding (`https://example.com ‚Üí https%3A%2F%2Fexample.com`)
  - Base64 Encoding (`https://example.com ‚Üí aHR0cHM6Ly9leGFtcGxlLmNvbQ==`)
- Deobfuscate URLs back to original
- CLI interface (no external libraries required)

---

## Use Cases

-  Analyze suspicious/phishing links
-  Practice decoding obfuscated malware URLs
-  Learn URL manipulation techniques in cybersecurity
